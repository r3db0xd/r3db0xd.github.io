# Cloud Architecture Portfolio

A high-performance, zero-dependency interactive portfolio and digital resume. 

As a Senior Cloud Systems Engineer, I wanted this site to reflect my engineering principles: lightweight, fast, and resilient. This project avoids heavy JavaScript frameworks in favor of pure, native web technologies.

## Technical Architecture
* **Core:** Pure HTML5 and Vanilla JavaScript.
* **Styling:** CSS variables for dynamic theme toggling (Dark Matrix / Light Cloud).
* **Physics Engine:** Custom CSS 3D isometric transforms and custom JS array shifting for the autonomous carousel, bypassing standard browser rendering artifacts.
* **Interactivity:** Decoupled 2D/3D event listeners to ensure perfect text rendering and zero Z-axis clipping.
